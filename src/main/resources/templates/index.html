Hello World!

I'm Martin (maddob) - a software engineer, fixed-gear and basketball fan. Always striving for simplicity

For a long time I was planning to start an own blog, but did not really know, what exactly to write about. After having discovered Vert.x a couple of months ago, I instantly fell in love with this marvellous framework. While experimenting with it, the idea of creating an own blog based on it and documenting the creation process was born - this is what you are reading at the moment. This page is powered by Vertx. Since all of my projects will be MAD, this one was named MadBlog. Feel free to check it out on github.

The first articles will give a detailed information about how this page was build. Since I am also a raspberry pi fan, the page you are reading at moment is hosted on my raspberry pi 3 at home! I was always wondering how much load can be handled by these marvellous machines and this page will be some sort of real case study. I will try to keep it running only on RPis if possible :) which will hopefully give me some more interesting information to share.
